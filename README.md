# 🎯 Popup Widget с валидацией согласия

Компонент модального окна для сайтов, решающий задачу **законного
перехода на внешний виджет** (Яндекс.Билеты, онлайн-кассы, партнёрские
сервисы). Реализует требование 152-ФЗ о согласии на обработку
персональных данных через чекбокс-валидацию.

> **Исходная задача:** «На сайт HTML нужно добавить попап окно: при
> открытии там должен быть чекбокс с согласием на обработку данных, и
> при нажатии кнопка становится активной. Кнопка — ссылка на
> Яндекс.Билеты. При нажатии окно закрывается и срабатывает переход».

## 🛠 Технологии

| Стек | Использование |
|------|---------------|
| **HTML5** | Семантическая разметка, `label`, `input[type=checkbox]`, атрибут `disabled` |
| **CSS3** | Flexbox, `position: fixed`, `inset`, утилитарный класс `.hidden` |
| **Vanilla JS (ES6+)** | `classList`, `addEventListener`, `localStorage`, `event.target` |
| **Git + GitHub Pages** | Версионирование и деплой без фреймворков |

❌ **Без зависимостей:** ни jQuery, ни Bootstrap, ни React. Чистый код, который встраивается в любой сайт за 5 минут.
